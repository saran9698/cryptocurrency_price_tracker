# cryptocurrency_price_tracker
A Python app to track real-time cryptocurrency prices and trends.
# Description

A Python application to fetch and track real-time cryptocurrency prices from APIs. Ideal for crypto enthusiasts, traders, and developers who want to analyze market trends.

## Features
- Track real-time prices of popular cryptocurrencies (Bitcoin, Ethereum, etc.)
- Get historical price data and trends
- Export data to CSV for analysis
- Interactive command-line interface

## Technologies Used
- Python 3.10
- Selenium
- Pandas (for data storage and manipulation)
- ChromeDriver
- Web-driverManager
